# ai-ru-kurer
🛵 AI Ru Kurer
Симулятор курьера в духе российских спальных районов. Доставляй борщ, шаурму и пиццу, уворачивайся от машин и зарабатывай рубли!

📖 О проекте
AI Ru Kurer — это браузерная аркадная игра, в которой ты играешь за курьера службы доставки еды в типичном российском городе. Бегай между панельными хрущёвками, разноси заказы из «Шаурмичной», «Бургерной» и «Пиццы Хат», следи за энергией и не попади под колёса!

Игра создана с помощью AI-инструментов и сочетает атмосферу постсоветских дворов с динамичным геймплеем.

✨ Особенности
🏙️ Атмосфера спальника — серые панельные дома, ночные фонари, пустые дворы
🍔 Реальные заказы — шаурма на углях, борщ домашний, бургеры и пицца
⚡ Система энергии — устал курьер, ползёт медленно. Купи энергетик в магазине!
🚗 Опасный трафик — следи за машинами, иначе доставка не состоится
💰 Экономика — зарабатывай рубли, прокачивайся, штрафы за просроченные заказы
💬 Живой курьер — реплики над головой («Холодно...», «Где этот подъезд?»)
🗺️ Процедурная карта — улицы, здания, перекрёстки и зелёные дворы
🛠️ Технологии
⚛️ React — компонентная архитектура UI
🔷 TypeScript — типобезопасность
🎨 TailwindCSS — стилизация интерфейса
🖼️ HTML5 Canvas — рендеринг игрового мира (через useRef)
🧊 Lucide React — иконки (рюкзак, деньги, еда)
🧠 Чистая игровая логика на JS/TS внутри React-компонентов (без Phaser и других движков)
🎮 Как играть
Управляй курьером по карте города
Получай заказы из ресторанов (жёлтые здания)
Доставляй еду клиентам в подъезды (зелёные зоны)
Следи за энергией и таймером доставки
Избегай красных машин на дорогах
Заходи в магазин за апгрейдами
🚀 Запуск
Bash

# Клонировать репозиторий
git clone https://github.com/yourname/ai-ru-kurer.git

# Перейти в папку
cd ai-ru-kurer

# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev
Открой http://localhost:5173 в браузере и начинай разносить заказы!

📸 Скриншоты
Ночной город, курьер в ярко-жёлтой форме с огромным рюкзаком пробирается между хрущёвками к очередному клиенту.
